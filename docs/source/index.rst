Welcome to FitCheck's documentation!
===================================

**Fitcheck** (/fɪt ʧɛk/) is an application aimed at users that want a place to improve and share their dress sense
or for those that just want a place to plan event outfits in advance.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
