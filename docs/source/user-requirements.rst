User Requirements
=================

These requirements will walk you through the general steps of using the FitCheck app. Below there will be a list of the requirements that will then lead you to where you can find the multiple steps to help you fulfil these requirements:

1. Be able to have an account on the app <requirement-1>
2. Be able to upload photos of their clothes <requirement-2>
3. Be able to organise items into categories <requirement-3>
4. Be able to create outfits <requirement-4>
5. Be able to receive recommendations on what outfits to wear according to the outing or event <requirement-5>
6. Be able to enter body measurements <requirement-6>
7. Be able to share or save outfits <requirement-7>
8. Be able to view trending outfits/styles <requirement-8>
