User Requirements
=================

These requirements will walk you through the general steps of using the FitCheck app. Below there will be a list of the requirements that will then lead you to where you can find the multiple steps to help you fulfil these requirements:

1. Be able to have an account on the app 
2. Be able to upload photos of their clothes 
3. Be able to organise items into categories 
4. Be able to create outfits 
5. Be able to receive recommendations on what outfits to wear according to the outing or event
6. Be able to enter body measurements 
7. Be able to share or save outfits
8. Be able to view trending outfits/styles
