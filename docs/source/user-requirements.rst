User Requirements
=================

These requirements will walk you through the general steps of using the FitCheck app. Below there will be a list of the requirements that will then lead you to where you can find the multiple steps to help you fulfil these requirements:

Requirement 1: Be able to have an account on the app
----------------------------------------------------

Requirement 2: Be able to upload photos of their clothes
--------------------------------------------------------

Requirement 3: Be able to organise items into categories
--------------------------------------------------------

Requirement 4: Be able to create outfits
----------------------------------------

Requirement 5: Be able to receive recommendations on what outfits to wear according to the outing or event
----------------------------------------------------------------------------------------------------------

Requirement 6: Be able to enter body measurements
-------------------------------------------------

Requirement 7: Be able to share or save outfits
-----------------------------------------------

Requirement 8: Be able to view trending outfits/styles
------------------------------------------------------
